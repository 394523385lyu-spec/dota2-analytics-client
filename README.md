# Dota 2 数据分析客户端

一个面向 Dota 2 战队和比赛公开数据的桌面分析工具。客户端基于 Python 和 PySide6，数据主要来自 OpenDota 公开接口，支持战队检索、近期比赛分析、BP 统计、选手表现、阵容共现和眼位可视化导出。

## 功能

- 按战队名称或 OpenDota Team ID 查询战队
- 获取当前选手、近期比赛、胜负、对手和赛事概况
- 将战队抓取结果导出为原始 `.xlsx`
- BP 选取/禁用、首末轮倾向与败局时间轴
- 选手近期表现、英雄使用和公开生涯档案
- 2–5 人阵容共现、胜率、英雄热度和共现加权度
- 眼位逐点明细、热力格点、存活质量与拆眼效率近似
- 专项结果可直接在客户端内按模块和数据表浏览、排序
- 眼位地图可在客户端内交互查看，也可另存为独立 HTML
- 客户端内眼位地图使用原生渲染，不依赖 Chromium/WebEngine
- 大数据表使用虚拟化模型，只绘制屏幕可见内容
- 眼位地图采用防抖、底图缓存和聚合热力渲染
- 比赛详情缓存 24 小时，重复分析和重启后无需重新下载
- 同一时间只运行一个重任务，避免 CPU、网络和内存相互争抢
- 英雄名称默认使用 Dota 2 官方简体中文，并兼容原汉化脚本的定制映射表
- 在本地完成字段识别、基础统计和数据压缩
- BP、选手档案、阵容共现、眼位分析分别导出独立 Excel
- 各工作簿保留原脚本对应的 Sheet 结构和表格样式
- 眼位可视化 HTML 显示天辉/夜魇、胜负、对手和时间阶段

## 环境要求

- Python 3.10 或更高版本
- macOS、Windows 或 Linux 桌面环境
- 可访问 OpenDota API 的网络环境

## 从源码运行

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Windows PowerShell 可使用：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

## 使用流程

1. 在“战队”页查询并载入战队。
2. 在“分析结果”页选择模块并开始分析。
3. 客户端内直接查看 BP、选手、阵容和眼位数据。
4. 在“导出文件”页分别导出各模块 Excel、原始数据 Excel 或眼位 HTML。

眼位分析中的“地图可视化”可切换比赛、双方、眼位类型、时间轴、消失状态以及点位/热力视图。

## 安全说明

- 原 Notebook 目录不会被修改。
- 运行日志保存在系统应用支持目录下，便于定位异常。
- 训练赛数据不纳入客户端。个人单排受公开数据和比赛可见性限制，不作为可靠分析模块。
- 眼位的“拆眼/分钟”沿用原脚本口径，是以对方眼位被提前移除数量计算的近似指标。
